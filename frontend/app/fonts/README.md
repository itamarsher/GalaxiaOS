# Inter (bundled)

`Inter.ttf` is the upright variable font from the Inter family
(https://github.com/rsms/inter), licensed under the SIL Open Font License 1.1
(OFL). It is bundled locally and loaded via `next/font/local` so the production
build needs no network access to Google Fonts. Inter is also the typeface used
in the ABOS logo, so the app type and the mark stay consistent.
