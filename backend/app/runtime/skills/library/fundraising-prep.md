# Fundraising Prep

Fundraising is selling the company's future on the strength of its real present. This playbook
assembles the narrative and materials — grounded in true numbers, because investors verify.

## Workflow
1. **Define the ask and use.** How much, for what runway, to hit which milestones? `read_financials`
   and `runway-and-burn-analysis` set a raise sized to reach a real inflection, not a round number.
2. **Assemble the real metrics.** Growth, retention, unit economics (`unit-economics-analysis`),
   burn, and pipeline — from measured data. Investors will diligence these; a single inflated
   number destroys credibility for all of them.
3. **Build the narrative.** Problem → why now → solution → traction → market → team → the plan for
   their money. Ground each claim in evidence; `draft_document` the deck outline (design it via
   `pitch-deck-design`).
4. **Prepare the data room.** Financials (`monthly-financial-close` outputs), metrics, cap table,
   key contracts. Organize (`save_file`) so diligence is fast and clean — friction reads as risk.
5. **Anticipate the hard questions.** List the weakest points honestly and prepare true answers.
   Investors respect founders who know their risks; spin is a red flag.
6. **Track the process.** Treat investors as a pipeline (`crm_save_deal`); `schedule_followup`;
   `write_memory` (type `learning`) objections that recur, to sharpen the story.

## Decision framework — honesty as strategy
Never inflate a metric or hide a risk. Diligence surfaces the truth, and getting caught ends the
raise. A modest number told straight beats an impressive one that unravels.

## Definition of done
- Ask sized to a real milestone; every metric from measured data and diligence-ready.
- Narrative evidence-backed; data room organized; hard questions answered honestly.

## Common failure modes
- **Inflated metrics.** One number that doesn't survive diligence poisons all of them.
- **Round-number asks.** Raise to a milestone, not to a vanity figure.
- **Hidden risks.** Investors find them anyway; owning them builds trust.
